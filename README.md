
# ⚠️ Nintendo 3DS Security Splash Collection

Repositorio dedicado a la recopilación y creación de **splashes de seguridad personalizados** para Nintendo 3DS.
Estos splashes aparecen al encender la consola antes del menú HOME.

---

## 📦 Contenido

* Splashes en formato compatible con Luma3DS
* Diseños personalizados (warnings, logos, imagenes)
* Versiones organizadas por estilo

---

## 📁 Estructura del repositorio

```
splashes/
│
├── warning/
├── minimalistas/
├── animados/
├── personalizados/
└── otros/
```

## ⚙️ Requisitos

Para usar estos splashes necesitas:

* Nintendo 3DS con **CFW (Custom Firmware)**
* Luma3DS actualizado

---

## 📥 Instalación

### Método 1: Splash automático (recomendado)

1. Descarga el splash que quieras
2. Renombra el archivo a:

```
splash.bin
```

3. Copia el archivo en:

```
/luma/
```

4. Enciende tu consola manteniendo presionado **SELECT**
5. Activa la opción:

* ✔️ `Show splash`

6. Guarda y reinicia
